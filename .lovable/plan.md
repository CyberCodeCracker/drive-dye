

# CovoitFacile — Site de Covoiturage

## Vue d'ensemble
Un site de covoiturage inspiré de BlaBlaCar, en français, avec une identité visuelle distincte utilisant des tons **vert émeraude (#10B981)** et **jaune doré (#F59E0B)** sur fond blanc/gris clair — un look frais et éco-responsable.

## Pages à créer

### 1. Page d'accueil
- **Header** : logo CovoitFacile, navigation (Rechercher, Publier, Connexion)
- **Hero section** : grande barre de recherche (départ, arrivée, date, nombre de passagers) sur une image de route/voyage
- **Section "Comment ça marche"** : 3 étapes illustrées (Recherchez, Réservez, Voyagez)
- **Trajets populaires** : cartes avec villes fréquentes
- **Footer** : liens utiles, réseaux sociaux

### 2. Résultats de recherche
- Filtres latéraux : heure de départ, prix max, préférences (fumeur, animaux, bagages)
- Liste de trajets : photo conducteur, note, prix, horaires, places restantes
- Tri par prix, heure, note

### 3. Détail d'un trajet
- Profil conducteur (photo, note, avis, véhicule)
- Itinéraire avec étapes intermédiaires
- Prix par passager, places disponibles
- Préférences du conducteur (musique, discussion, fumeur)
- Bouton "Réserver"

### 4. Publier un trajet
- Formulaire : départ, arrivée, date/heure, prix, places, étapes intermédiaires
- Préférences conducteur
- Aperçu avant publication

### 5. Page de réservation
- Récapitulatif du trajet
- Choix du nombre de places
- Message au conducteur
- Confirmation de réservation

### 6. Dashboard Admin
- Statistiques (trajets, utilisateurs, réservations)
- Liste des trajets avec modération
- Gestion des utilisateurs
- Graphiques d'activité

## Design
- **Couleurs** : Vert émeraude primaire, jaune doré pour les accents/CTA, gris foncé pour le texte
- **Style** : cartes arrondies, ombres douces, typographie claire et moderne
- **Responsive** : adapté mobile et desktop
- **Données** : données fictives réalistes (villes françaises, noms, prix en €)
